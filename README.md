# AJCC-Stage-form-Autohotkey-V1.1
以AutoHotKey v1.1寫成的腳本
19項癌症分期表單的GUI，用以生成純文字(ANSI)報告

如

![這是示範圖](/pancreas00.jpg)

勾選後生成

![這是示範圖](/pancreas01.jpg)

純文字報告到剪貼簿，可以直接到報告欄位按ctrl+v

>[!warning]
腳本在Github中編碼可能會被改為UTF-8，
下載後須手動修改為ANSI或Big-5
